Possible FPS gains

Some users have reported that running this script increased their FPS.
Warning: The script UNTRACKS ALL TRACKED QUESTS including hidden tracked quests

The script is run directly in your chat box

Link to Reddit Post

Script:

/run for i=1, 200000 do C_QuestLog.RemoveQuestWatch(i) end

Core Addons

BugGrabber

Must Have
  • Grabs error messages, prevent error frame appearing in the middle of gameplay.
  • Used together with BugSack
  • Must have because the Blizzard default UI itself still throws a lot of errors randomly

BugSack

Must Have
/bugsack show
  • Display errors in previous or current game session.
  • Use this together with BugGrabber
  • Must have because the Blizzard default UI itself still throws a lot of errors randomly
Screenshots

Simple Addon Manager

Recommended
Replaces System Menu -> Addons
  • Better display of addons accessible via the Main Menu  `¤→ Addons button.
  • Removes all annoying addon categories
  • Displays addon CPU usage to easily find addons that are clogging the game
  • Create profiles of selected addons for quick and easy reloads
Screenshots

Frames

BetterBlizzFrames

Recommended with base frames
/bbf
  • Adds ability to customize the default Blizzard Unit/Party frames.
  • Buff filtering on player, party castbars, class color unit frames, etc.
  • Lots of settings to tweak and customize, and comes with some very good default profiles based on popular streamers' choices
  • Low to no CPU usage
Screenshots

FrameColor - Dark Mode & Custom Color UI

/fc
  • Applies a "dark mode" skin to all UI elements which is easy on the eyes.
  • Adds ability to customize colors for all UI elements (think menus, bars, calendar, character pane, anything) with your class color or color of choice
  • Very good "out of the box" settings for a darker feel to the UI without using ElvUI Dark Mode
  • Low to no CPU usage

FrameColor - AddOnSkins

  • Provides the ability to customize some external addons frames, including FrameColor (lol).

HealthBarColor - Health Bar Color

/hbc
  • Adds color customizations to Class Colors, Reaction Colors, Power Colors, Health Colors.
  • Change texture on Health bars and Power bars
  • Modules to change other stuff, including textures, fonts, background textures, glow, party colors
  • Low to no CPU usage
  • Luxthos Profile can be found here

Cooldown Manager (Class Based WeakAura's Replacements)

Note 1: Enable Cooldown Manager in Settings  `¤→ Gameplay  `¤→ Gameplay Enhancements  `¤→ Cooldown Manager  `¤→ ✅ Enable Cooldown Manager
Note 2: Luxthos has free and readily available profiles for the Cooldown Manager on his website for all classes

Cooldown Manager Centered

/cdmc
  • Adds directional alignment support to the Cooldown Manager
  • Adds various styling options for the cooldown manager to customize look-and-feel
  • Adds ability to show Rotation Highlight on CDM (one button rotation assist highlight)
  • Adds ability to synchronize the width of castbar to your Essential Icons bar
  • Luxthos Profile can be found here
Screenshots

SenseiClassResourceBar

  • Lightweight, fully customizable resource display addon
  • Shows primary and secondary resources
  • Supports "ticks" for some classes
  • Auto-changing of resources for Druids based on form
  • Syncs width of resource bar with Cooldown Manager
  • Luxthos Profile can be found here
Screenshots

Combat/Raid/M+ Addons

Details! Damage Meter

Recommended
/details options
  • Reskinning of Blizzard UI for Damage Meter
  • Same functionality as Blizzard's own Damage Meter, but much better looking and more customizable
Screenshots

DBM - Deadly Boss Mods (DBM-Core)

Recommended without BigWigs
/dbm
  • Customizes the Blizzard UI for Boss Timers and Alerts, as well as adding sound packs

BigWigs Encounter Tools

Recommended without DBM
/bw
  • Same as DBM, only for Raid Encounters - just in another package with different defaults. Use the one you like the best

LittleWigs

Recommended with BigWigs
  • BigWigs/DBM functionality for dungeons
  • Requires BigWigs to function
  • Options in interface for BigWigs

Plater Nameplates

Recommended
/plater
  • Plater is a nameplate addon with a extraordinary amount of settings
  • Better than default nameplates (opinion)
  • Supports profiles obtainable via creators and https://wago.io/plater
  • Jundies - Midnight & Pre-Patch M+ and Raid Plater is a very good profile
Screenshots

Platynator

Recommended
/platynator
  • Platynator is a simple nameplate addon with a good amount of settings and customizability. Simpler to use than Plater, but also has less features.
  • Better than default nameplates (opinion)
  • Luxthos Profile can be found here

Missing Class Buff

Recommended
/mcb
  • Notifies you when you are missing important class buffs, flasks, food, weapon oils, or other trackable buffs, customizable by id.
  • Can be customized to display in different content, such as only in raids, only in dungeons, etc.
  • Very useful for raid and dungeon content so you don't forget to bring that extra damage
  • Customizable buffs to track per class with sane defaults
  • Can be set to clicking the frame will apply the buff/food/flask/etc
Screenshots

MPlusTimer

Recommended
/mplustimer
  • The M+ Timer WeakAura now in addon form
  • Displays all the information you need about your current M+ run in a nicer and more modern way than the default UI does
Screenshots

Keystone Polaris (formerly Keystone Percentage Helper)

Recommended
/kpl
  • Assists you in tracking enemy forces percentage during Mythic+ dungeon runs

Leatrix Maps

Plumber

Recommended Must Have
/plumber
  • Extremely well done quality of life addon that will make your WoW experience better without you even knowing it was an addon doing it.
  • Faster auto-looting and a pretty loot frame for looted items
  • Improvements to the talking head frame
  • Create drawer-macros for easy access to frequently used macros
  • Enhanced Expansion Summary UI with everything you need in one place
  • Instance Difficulty Selector when you are at the entrance of a raid or dungeon
  • Many more features, and perhaps one of the more beautifully designed addons out there

Edit Mode Expanded

Recommended
Adds to the Edit Mode experience
  • Allows you to move more frames than the default Edit Mode experience
  • Adds various customization options for frame positioning
  • Adds various options to hide frames when in certain situations
  • Resize custom frames
