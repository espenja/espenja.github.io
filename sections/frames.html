BetterBlizzFrames

Recommended with base frames
/bbf
  • Adds ability to customize the default Blizzard Unit/Party frames.
  • Buff filtering on player, party castbars, class color unit frames, etc.
  • Lots of settings to tweak and customize, and comes with some very good default profiles based on popular streamers' choices
  • Low to no CPU usage

FrameColor - Dark Mode & Custom Color UI

/fc
  • Applies a "dark mode" skin to all UI elements which is easy on the eyes.
  • Adds ability to customize colors for all UI elements (think menus, bars, calendar, character pane, anything) with your class color or color of choice
  • Very good "out of the box" settings for a darker feel to the UI without using ElvUI Dark Mode
  • Low to no CPU usage

HealthBarColor - Health Bar Color

/hbc
  • Adds color customizations to Class Colors, Reaction Colors, Power Colors, Health Colors.
  • Change texture on Health bars and Power bars
  • Modules to change other stuff, including textures, fonts, background textures, glow, party colors
  • Low to no CPU usage
