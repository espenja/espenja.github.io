DBM - Deadly Boss Mods (DBM-Core)

Recommended without BigWigs
/dbm
  • Customizes the Blizzard UI for Boss Timers and Alerts, as well as adding sound packs

BigWigs Encounter Tools

Recommended without DBM
/bw
  • Same as DBM, only for Raid Encounters - just in another package with different defaults. Use the one you like the best

LittleWigs

Recommended with BigWigs
  • BigWigs/DBM functionality for dungeons
  • Requires BigWigs to function
  • Options in interface for BigWigs

Plater Nameplates

Recommended

  • Better than default nameplates (opinion)
  • Supports profiles obtainable via creators and https://wago.io/plater
  • Jundies - Midnight & Pre-Patch M+ and Raid Plater is a very good profile

Platynator

Recommended
/platynator
  • Platynator is a simple nameplate addon with a good amount of settings and customizability. Simpler to use than Plater, but also has less features.
  • Better than default nameplates (opinion)
  • Luxthos Profile can be found here
Screenshots

Missing Class Buff

Recommended
/mcb
  • Notifies you when you are missing important class buffs, flasks, food, weapon oils, or other trackable buffs, customizable by id.
  • Can be customized to display in different content, such as only in raids, only in dungeons, etc.
  • Very useful for raid and dungeon content so you don't forget to bring that extra damage
  • Customizable buffs to track per class with sane defaults
  • Can be set to clicking the frame will apply the buff/food/flask/etc
Screenshots
