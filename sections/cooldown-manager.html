Cooldown Manager (Class Based WeakAura's Replacements)

Note 1: Enable Cooldown Manager in Settings → Gameplay → Gameplay Enhancements → Cooldown Manager → ✅ Enable Cooldown Manager
Note 2: Luxthos has free and readily available profiles for the Cooldown Manager on his website for all classes

Cooldown Manager Centered

/cdmc
  • Adds directional alignment support to the Cooldown Manager
  • Adds various styling options for the cooldown manager to customize look-and-feel
  • Adds ability to show Rotation Highlight on CDM (one button rotation assist highlight)
  • Adds ability to synchronize the width of castbar to your Essential Icons bar
  • Luxthos Profile can be found here

SenseiClassResourceBar

  • Lightweight, fully customizable resource display addon
  • Shows primary and secondary resources
  • Supports "ticks" for some classes
  • Auto-changing of resources for Druids based on form
  • Syncs width of resource bar with Cooldown Manager
  • Luxthos Profile can be found here
